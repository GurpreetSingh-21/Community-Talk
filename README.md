# Community Talk

A fun and clean web app where students from different colleges can join their own communities, chat, and stay connected.

### Features so far:
- 🔐 User registration and login
- 🧠 Login validation with backend (Node.js + MongoDB)
- 👥 Community-based chat UI (Queens, Baruch, Hunter)
- ✅ Responsive design with sidebar and mobile support
- 🔄 Logout functionality
- 💬 Members list and message display

More features coming soon!
